What this is

Same Claude Code, different brain

Claude Code is Anthropic's terminal AI coder. By default it talks to Anthropic's models. cc-switcher flips a few environment variables so Claude Code talks to another provider instead — then restores everything when you quit. Same interface, cheaper (or free) brain.

The #1 thing to understand

Two ways you pay

Every provider uses one (or both) of these. As a beginner, start pay-per-token to learn what things actually cost, then switch to a subscription once you know your daily usage.

💧 Pay-per-token (metered)

Prepay credits (e.g. $10). Each message costs a few cents. Pay only for what you use.

Best for: experimenting, light or spiky use.

Watch out: a huge job can cost more than expected — set a credit cap.

Examples: OpenRouter, DeepSeek, MiniMax/Kimi/Xiaomi PAYG.

📅 Subscription ("coding plan")

Fixed price/month (e.g. $18). Use as much as the quota allows. No math.

Best for: coding daily, wanting a predictable bill.

Watch out: hit the quota → wait for the rolling reset (often every 5 hrs).

Examples: Z.AI GLM, MiniMax Token Plan, Kimi Code, Ollama, OpenCode Go, ChatGPT.

🧠 Prices are per 1 million tokens (a token ≈ ¾ of a word). Output tokens (what the AI writes) cost 2–5× more than input — and coding agents write a lot, so the output price is the number that matters most.

⭐ The fastest start

OpenRouter — one key, nine providers

OpenRouter is a universal adapter: one account and one key unlock hundreds of models. In cc-switcher, a single OPENROUTER_API_KEY powers nine provider commands (plus the generic cc-openrouter) — so you sign up once and try almost the whole catalog.

cc-glmcc-geminicc-grok cc-kimicc-mimocc-qwen cc-minimax-orcc-nemotron free cc-owl freecc-openrouter <model>

Sign up & get your key (5 min)

  1. Register at openrouter.ai/sign-up (Google / GitHub / email).
  2. Add credits at settings/credits. 💡 Buy at least $10 — it lifts the free-model limit from 50 → 1,000 req/day and never expires. (Card adds ~5.5% fee.)
  3. Create a key at settings/keys → it looks like sk-or-v1-…. Copy it now — shown once.
  4. Add to your shell profile and reload:
# bash / zsh
export OPENROUTER_API_KEY="sk-or-v1-..."
# PowerShell
$env:OPENROUTER_API_KEY = "sk-or-v1-..."

cc-doctor        # verify
cc-glm           # GLM-5.2 — great all-round coder, 1M context
cc-nemotron      # FREE — zero-risk first run
OpenRouter is pay-per-token only — no monthly plan. The :free models (cc-nemotron, cc-owl) cost nothing but are rate-limited.

Tips for someone brand-new to AI

  1. Keep API keys secret. They cost money. Put them in your shell profile, never in code or git.
  2. Set a spending cap on OpenRouter (per-key credit limit). Prepaid credits are your safety net.
  3. Start cheap, taste-test. Run the same prompt through cc-deepseek, cc-glm, cc-qwen — they're cheap enough to compare freely.
  4. Use cc-launch if you forget command names — it's a menu.
  5. Make beginner mistakes on free models (cc-nemotron) where they cost nothing.
  6. Run cc-doctor when something fails — it tells you if a key's missing or the endpoint's down.
  7. You can always cc-reset back to normal Claude. Your shell is never left in a weird state.
  8. Check cc-usage after a few sessions to see real spend — it demystifies cost fast.
